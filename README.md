# OpenGL 

---

## Description 
Creating 3D and 2D shapes using OpenGL and Glut. 

3D objects: 
- Cube
- Pyramid
- Cone
- Cyllinder
- Icosahedron
- Tetrahedron
- Dodecahedron
- Octahedron
- Thorus
- Sphere

2D objects:
- Square
- Triangle
- Letters A, K, T

Used libraries:
```c#
using SharpGL;
using Tao.OpenGl;
using Tao.FreeGlut;
```
